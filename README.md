## Project Description

This project was developed as part of the Fundamentals of Computing course unit. The objective is to recreate the design, layout, and basic user interface functionality of a publicly accessible website using only HTML and CSS.

The project demonstrates our understanding of web development concepts such as page structure, styling, responsiveness, navigation, and user interface design.

## Original Website

**Website Name:** Agakhan Health Services

**Original URL:** [Click to view Website](https://agakhanhospitals.org/en/home)

## Project Objectives

* Understand the structure of modern websites.
* Practice creating web pages using HTML.
* Apply CSS styling techniques to replicate layouts and visual designs.
* Develop responsive web pages that adapt to different screen sizes.
* Improve teamwork and collaborative development skills.
